# 2d-Mincraft-moder

THIS IS FOR USE WITH https://github.com/python-nerd1235/2d-Minecraft-with-Pygame. THIS IS NOT A STANDALONE GAME.

TO MAKE A MOD:

1.  run the program.
2.  enter the name of your block
3.  enter the couler of your arm when the block is selected(Red, green, blue).
4.  enter the name of the texture file (like grass.png).
5.  if you whant to add more to your mod, then type N. if you whant to stop, type Y. if you typeed Y, go to step 2.
6.  enter the name of your mod.

HAVE FUN!
